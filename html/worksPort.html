Currently, I am working on a project called ERDATA which is web-based system software that helps helpers to view the children's photos with their information such as name, age, and residence, and allows users to select the photo of their preference to go to a profile page of the selected child and it will give the choices for donation(help).

I have developed an Online clearance system which is a web-based platform that allows the student to ask for clearance approval after login into the system using their username and password. The ultimate goal of this product is to change the present manually done clearance paper giving process to an online system which saves time, effort, cost, etc required for this process.

I have developed Snap CSS, a tool that optimizes CSS into a more professional form by analyzing the actual CSS content to make it more consistent and in agreement with industry standards. Snap CSS will provide a script that performs the following functions: Scan CSS styles from a source code, Group styles with the same properties, Remove redundant CSS, Convert longhand notations to shorthand notations, Consolidate multiple value representations into a single one, and Suggest standard class names
